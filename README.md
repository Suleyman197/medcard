# MedicalCard Security — Caesar Cipher (C++)

Данную работу выполнял: Бахалов Сулейман
Данный репозиторий создан в рамках СРС 1  
**«Защита системы и данных в мобильных приложениях»**

## 📌 Описание
Реализован простой механизм защиты пароля пользователя
перед сохранением с использованием классического шифра Цезаря.
Готовые криптографические библиотеки не использовались.

## ⚙️ Используемый алгоритм
- Шифр Цезаря
- Ключ сдвига: `3`
- Язык программирования: `C++`
